What the Network fit labels mean.

Each label answers one question: could a network signal have changed a documented decision before the attacker succeeded?

Direct intervention

A documented network event maps to the exact decision point before the attacker succeeds.

Supporting signal

Network context could improve risk scoring or step-up, but it would not stop the attack by itself.

No material impact

The path centers on vulnerabilities, service credentials, sessions, or infrastructure outside mobile identity.

Unknown

The public record does not support a responsible mapping. Repeated reporting is not a substitute for evidence.
